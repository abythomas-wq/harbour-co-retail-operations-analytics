# SwiftFulfil Operations Analytics

## Project Overview

SwiftFulfil is a fictional e-commerce fulfilment company. This project analyses operational data to identify opportunities for improving business performance, efficiency, and decision-making.

The objective is to simulate a real-world analytics environment where data from customers, products, suppliers, orders, and fulfilment operations is transformed into actionable business insights.

## Business Objectives

The analysis aims to answer key operational questions:

- Which products and categories generate the highest revenue and profit?
- Which suppliers perform best and which require improvement?
- What customer segments contribute the most value?
- How can operational efficiency be improved?
- Where are potential risks in the supply chain?

## Tools & Technologies

- Python - Data generation and data preparation
- MySQL - Database design and SQL analysis
- SQL - Data querying and business analysis
- GitHub - Version control and portfolio management
- Power BI - Data visualisation and dashboard development
